# 🦞 ClawGuard / 龙虾卫士

AI-era digital asset proof system for creators and digital workers.

## Quick Reference

| Command | Description |
|---------|-------------|
| `龙虾卫士，AI 确权` | Prove AI-generated content |
| `龙虾卫士，生成凭证` | Generate proof certificate |
| `龙虾卫士，验真` | Verify proof authenticity |
| `龙虾卫士，查看记录` | List all proofs |
| `龙虾卫士，导出记录` | Export proof list |
| `龙虾卫士，批量确权` | Batch prove folder |
| `龙虾卫士，搜索` | Search proofs |
| `龙虾卫士，统计` | Get statistics |

## Features

- ✅ AI Proof (image/video/audio/code/document)
- ✅ Multi-language certificates (CN/EN/Bilingual)
- ✅ Multi-owner support
- ✅ Local blockchain (hash-linked)
- ✅ NTP timestamp (ISO 8601)
- ✅ Batch proof (folder)
- ✅ Search & stats
- ✅ Offline-first, privacy protection
