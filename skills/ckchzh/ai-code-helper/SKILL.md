# Claude Code

Claude Code is an agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster by executing routine tasks, explaining complex code, and handling git workflows - all through natural language commands. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from anthropics/ai-code-helper

## Usage

Run any command: `ai-code-helper <command> [args]`

## Examples

```bash
# Show help
ai-code-helper help

# Run
ai-code-helper run
```

- Run `ai-code-helper help` for commands
- No API keys needed

- Run `ai-code-helper help` for all commands

## Configuration

Set `AI_CODE_HELPER_DIR` to change data directory. Default: `~/.local/share/ai-code-helper/`
