# 行政法全流程AI助手 🏛️

> 专业行政法律师AI助手 | 行政复议 | 行政诉讼 | 政府信息公开 | 行政赔偿

## 🎯 适用场景

- 行政处罚复议/诉讼
- 行政许可争议
- 行政强制纠纷
- 政府信息公开申请
- 行政赔偿/补偿
- 征地拆迁纠纷
- 环保处罚应对
- 工商税务争议

## 📋 阶段一：案件评估与路径选择

### 1.1 行政行为合法性初步判断
```
你是一位资深行政法律师。请对以下行政行为进行合法性初步判断：

行政行为：[处罚决定/许可决定/强制措施/等]
作出机关：[机关名称]
行政相对人：[当事人]
行为内容：[具体内容]
作出时间：[日期]

请分析：
1. 主体合法性（是否有管辖权）
2. 程序合法性（是否履行法定程序）
3. 实体合法性（事实认定、法律适用）
4. 明显违法点识别
5. 维权路径建议（复议/诉讼/投诉）
```

### 1.2 复议vs诉讼路径选择
```
请分析以下案件，建议最优维权路径：

案件类型：[行政处罚/行政许可/行政强制/等]
当事人诉求：[撤销/变更/确认违法/赔偿]
时间因素：[行为作出时间]

路径分析：
1. 复议前置情形判断
2. 复议机关确定
3. 诉讼管辖确定
4. 时间成本对比
5. 效果预期对比
6. 最优路径建议
```

### 1.3 被告/被申请人确定
```
请确定以下案件的适格被告/被申请人：

行政行为：[行为描述]
作出机关：[机关名称]
涉及机关：[多个机关情况]

分析要点：
1. 职权主体识别
2. 委托执法情形
3. 共同被告情形
4. 经复议后的被告确定
5. 派出机构/内设机构情形
```

### 1.4 起诉期限/复议期限计算
```
请计算以下案件的起诉期限/复议期限：

行政行为：[行为描述]
送达时间：[日期]
知道时间：[日期]
行为类型：[作为/不作为]

计算内容：
1. 复议期限（60日）
2. 直接起诉期限（6个月）
3. 经复议后起诉期限（15日）
4. 最长保护期限
5. 期限届满日
6. 是否已超期及补救措施
```

## 📋 阶段二：行政复议

### 2.1 行政复议申请书
```
请起草行政复议申请书：

申请人：[姓名/单位]
被申请人：[机关名称]
复议请求：[撤销/变更/确认违法/责令履行]

申请书应包含：
1. 申请人信息
2. 被申请人信息
3. 复议请求（明确具体）
4. 事实与理由
   - 程序违法点
   - 实体违法点
   - 证据分析
5. 法律依据
6. 证据清单
```

### 2.2 复议答辩书（被申请人）
```
请起草行政复议答辩书：

答辩机关：[机关名称]
复议申请人：[姓名]
复议请求：[请求内容]

答辩书应包含：
1. 答辩机关信息
2. 对复议请求的回应
3. 事实认定说明
4. 程序合法性说明
5. 法律适用说明
6. 证据支持
7. 答辩结论
```

### 2.3 复议决定分析
```
请分析以下行政复议决定：

[粘贴复议决定]

分析维度：
1. 决定类型评价（维持/撤销/变更/确认违法）
2. 事实认定评价
3. 法律适用评价
4. 程序合法性评价
5. 后续路径建议
   - 接受决定
   - 提起行政诉讼
   - 申请国务院裁决
```

## 📋 阶段三：行政诉讼

### 3.1 行政起诉状
```
请起草行政起诉状：

原告：[姓名/单位]
被告：[机关名称]
诉讼请求：[撤销/变更/确认违法/履行/赔偿]

起诉状应包含：
1. 当事人信息
2. 诉讼请求（明确具体）
3. 事实与理由
   - 行政行为描述
   - 违法点分析
   - 损害后果
4. 证据清单
5. 此致法院
```

### 3.2 行政答辩状
```
请起草行政答辩状：

答辩机关：[机关名称]
原告：[姓名]
诉讼请求：[请求内容]

答辩状应包含：
1. 答辩人信息
2. 对诉讼请求的回应
3. 行政行为合法性说明
4. 证据支持
5. 答辩结论
```

### 3.3 行政赔偿诉讼
```
请起草行政赔偿起诉状：

赔偿请求人：[姓名]
赔偿义务机关：[机关名称]
赔偿请求：[金额/方式]

起诉状应包含：
1. 赔偿请求人信息
2. 赔偿义务机关信息
3. 赔偿请求（具体金额/方式）
4. 损害事实
5. 因果关系说明
6. 赔偿依据
7. 证据清单
```

## 📋 阶段四：政府信息公开

### 4.1 信息公开申请
```
请起草政府信息公开申请：

申请人：[姓名/单位]
申请机关：[机关名称]
申请内容：[信息描述]

申请书应包含：
1. 申请人信息
2. 申请公开的信息内容
   - 信息名称
   - 信息描述
   - 信息用途
3. 信息载体形式（纸质/电子）
4. 获取信息方式
5. 法律依据
```

### 4.2 信息公开答复分析
```
请分析以下信息公开答复：

[粘贴答复内容]

分析维度：
1. 答复类型（公开/不予公开/部分公开/信息不存在）
2. 答复期限合规性
3. 不予公开理由合法性
4. 救济途径提示
5. 后续行动建议
```

## 📋 阶段五：征地拆迁专项

### 5.1 征收决定审查
```
请审查以下征收决定的合法性：

征收项目：[项目名称]
征收机关：[机关名称]
征收范围：[范围描述]

审查要点：
1. 征收主体合法性
2. 征收程序合法性
3. 补偿标准合理性
4. 评估程序合法性
5. 违法点识别
6. 维权建议
```

### 5.2 补偿安置方案分析
```
请分析以下补偿安置方案：

[粘贴补偿方案]

分析维度：
1. 补偿标准合法性
2. 安置方式合理性
3. 评估价值公允性
4. 与周边项目对比
5. 协商策略建议
6. 诉讼准备要点
```

## 📋 阶段六：环保处罚专项

### 6.1 环保处罚应对
```
请制定环保处罚应对策略：

处罚机关：[生态环境局]
处罚事由：[违法事实]
处罚内容：[罚款金额/停产整治等]

应对策略：
1. 处罚程序审查
2. 证据链分析
3. 法律适用审查
4. 量罚合理性分析
5. 复议/诉讼可行性
6. 整改方案制定
7. 协商和解策略
```

## 📋 阶段七：工商税务专项

### 7.1 税务处罚应对
```
请制定税务处罚应对策略：

处罚机关：[税务局]
处罚事由：[偷税/漏税/虚开发票等]
处罚金额：[金额]

应对策略：
1. 税务稽查程序审查
2. 事实认定分析
3. 税款计算复核
4. 滞纳金/罚款合理性
5. 首违不罚适用
6. 分期缴纳申请
7. 复议/诉讼策略
```
