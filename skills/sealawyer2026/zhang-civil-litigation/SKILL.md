# 民商事诉讼全流程AI助手 ⚖️

> 专业民商事律师的AI助手 | 7大阶段 | 15个模板 | 全流程覆盖

## 🎯 适用场景

- 民事案件评估与接案
- 起诉/应诉准备
- 证据收集与整理
- 庭前准备与策略制定
- 庭审应对与辩论
- 判决分析与上诉
- 执行阶段代理

## 📋 阶段一：案件评估

### 1.1 案件可行性分析
```
你是一位资深民商事律师。请对以下案件进行可行性分析：

案件类型：[合同纠纷/侵权纠纷/婚姻家庭/劳动争议/等]
当事人诉求：[具体诉求]
现有证据：[证据清单]
对方情况：[已知信息]

请提供：
1. 胜诉概率评估（百分比）
2. 主要风险点
3. 关键证据缺口
4. 诉讼策略建议
5. 费用预算
6. 时间周期预估
```

### 1.2 管辖权分析
```
请分析以下案件的管辖权问题：

案件性质：[合同/侵权/不动产/等]
当事人所在地：[地点]
合同履行地：[地点]
标的物所在地：[地点]

分析内容：
1. 法定管辖法院
2. 协议管辖有效性（如有）
3. 专属管辖适用
4. 管辖权异议策略
5. 建议管辖法院
```

## 📋 阶段二：立案准备

### 2.1 起诉状生成
```
请根据以下信息起草民事起诉状：

原告：[姓名/单位]
被告：[姓名/单位]
案由：[具体案由]
诉讼请求：[1. 2. 3.]
事实与理由：[简要案情]

起诉状应包含：
1. 当事人信息（完整）
2. 诉讼请求（明确具体）
3. 事实与理由（逻辑清晰）
4. 证据清单
5. 此致法院
```

### 2.2 答辩状生成
```
请根据以下信息起草民事答辩状：

案号：[案号]
原告：[姓名]
被告：[姓名]
案由：[案由]
原告主张：[简要内容]

答辩状应包含：
1. 答辩人信息
2. 针对各项诉讼请求的答辩意见
3. 事实与理由
4. 证据支持
5. 反诉请求（如有）
```

### 2.3 证据清单制作
```
请制作证据清单：

案件：[简要案情]
证明目的：[需要证明的事实]

清单格式：
| 序号 | 证据名称 | 证据来源 | 证明内容 | 页码 |
|-----|---------|---------|---------|-----|
| 1   |         |         |         |     |

补充建议：
1. 缺失证据提示
2. 证据补强建议
3. 证据链完整性检查
```

## 📋 阶段三：证据整理

### 3.1 证据目录编排
```
请编排以下证据的证据目录：

[证据列表]

编排原则：
1. 按证明目的分类
2. 按时间顺序排列
3. 重要证据前置
4. 每组证据加小标题
5. 编制页码索引
```

### 3.2 质证意见准备
```
请准备对对方证据的质证意见：

对方证据：[证据名称]
证据内容：[简要描述]

质证要点：
1. 真实性异议（如有）
2. 合法性异议（如有）
3. 关联性异议（如有）
4. 证明力评价
5. 质证结论
```

## 📋 阶段四：庭前准备

### 4.1 庭审提纲制作
```
请制作庭审提纲：

案件：[简要案情]
争议焦点：[焦点1/焦点2/焦点3]
我方立场：[立场]

提纲内容：
1. 开场陈述要点
2. 举证顺序与重点
3. 质证策略
4. 法庭询问预案
5. 辩论要点
6. 最后陈述
```

### 4.2 代理词撰写
```
请撰写代理词：

案件：[简要案情]
代理方：[原告/被告/第三人]
代理思路：[思路]

代理词结构：
1. 首部
2. 案件事实陈述
3. 法律适用分析
4. 争议焦点论证
5. 代理意见总结
6. 尾部
```

## 📋 阶段五：庭审应对

### 5.1 法庭辩论要点
```
请整理法庭辩论要点：

争议焦点：[焦点]
我方观点：[观点]
对方观点：[观点]

辩论要点：
1. 事实认定层面
2. 法律适用层面
3. 证据采信层面
4. 程序合法性层面
5. 反驳对方观点
```

### 5.2 突发情况应对
```
请提供以下突发情况的应对策略：

情况：[对方突然提交新证据/法官提问超出预期/等]

应对策略：
1. 即时反应话术
2. 法律依据引用
3. 程序权利主张
4. 后续补救措施
```

## 📋 阶段六：判决分析

### 6.1 判决书分析
```
请分析以下判决书：

[粘贴判决书主要内容]

分析维度：
1. 判决结果评价
2. 事实认定分析
3. 法律适用评价
4. 证据采信分析
5. 上诉可行性评估
6. 上诉策略建议
```

### 6.2 上诉策略制定
```
请制定上诉策略：

一审判决：[结果]
上诉期限：[截止日期]
当事人意愿：[是否上诉]

策略内容：
1. 上诉理由选择
2. 新证据准备
3. 法律适用纠正
4. 二审预期结果
5. 风险评估
```

## 📋 阶段七：执行阶段

### 7.1 执行申请书
```
请起草执行申请书：

生效文书：[判决书/调解书]
案号：[案号]
被执行人：[姓名/单位]
执行标的：[金额/行为]

申请书应包含：
1. 申请人信息
2. 被执行人信息
3. 执行依据
4. 执行请求
5. 财产线索（如有）
```

### 7.2 执行异议申请
```
请起草执行异议申请书：

执行案件：[案号]
异议人：[身份]
异议事项：[具体事项]

申请书应包含：
1. 异议人信息
2. 执行案件信息
3. 异议请求
4. 事实与理由
5. 法律依据
```
