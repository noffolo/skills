# TJWeather Skill (tjweather) 🌤️

这是一个为 OpenClaw 量身打造的全球天气查询插件，核心调用 **TJWeather API**。它不仅支持精准的全球天气预测，还内置了三级地理编码回退机制，确保在各种网络环境下都能稳定获取位置坐标。

## 🌐 官方资源
- **气象官网**: [https://www.tjweather.com/](https://www.tjweather.com/)
- **API Key 申请**: [https://www.tjweather.com/Apply](https://www.tjweather.com/Apply)

## 🛠️ 环境配置要求
- **运行环境**: Python 3.8+ 
- **依赖库**: 仅需 Python 标准库 (`sys`, `json`, `urllib`, `difflib`)，无需额外安装第三方包。
- **网络能力**: 
  - **方案一/二 (首选)**: 需要具备**国际网络访问能力**（以连接 OpenStreetMap 等国际地理编码服务）。
  - **方案三 (离线模式)**: 内置中国行政区划数据库，可在无国际网络时作为大陆地区的兜底方案。

## 🔑 Key 的配置
本 Skill 依赖 `TJWEATHER_API_KEY` 进行身份验证。该 Key 必须配置在 Skill 根目录下的 **`.env`** 文件中。**该 Key 可在上方官方资源链接中申请。**

### 如何找回 API Key
请按以下步骤在天机官方网站找回 API 密钥（API Key）：
1. 访问 https://www.tjweather.com/vis/ 并点击右上角的**【登录/注册】**完成登录。
2. 登录后，点击同样位置的**【菜单】**图标，进入**【个人中心】**。
3. 在个人中心页面中选择**【API管理】**即可查看或重置您的 API Key。

### 配置步骤
1. 在 `~/.openclaw/skills/tjweather/` 目录下找到或创建 `.env` 文件。
2. 写入以下内容：
   ```env
   TJWEATHER_API_KEY=您的_TJWEATHER_API_KEY
   ```
*注：Agent 会自动读取该配置，无需向用户直接索要 API Key。*

## 📍 地理编码查询 (3种回退方案)
为了保证全球范围内搜索的准确性与稳定性，本工具按优先级依次尝试以下三种方法：

1. **方案一 (首选 - Nominatim)**: 调用 OpenStreetMap (OSM) 数据。该方法具备最高精度的全球地名识别能力，但**必须要求系统具有国际网络访问能力**。
2. **方案二 (备用1 - Photon)**: 基于 Komoot 的搜索引擎。作为首选方案失效后的快速补充，同样依赖国际网络。
3. **方案三 (备用2 - 中国内部库)**: **中国大陆兜底方案**。内置了全量中国省级、市级行政区划的中心点坐标。当网络受限或查询国内偏远城市时，该方案能确保提供稳定、快速的坐标匹配。

## 📝 使用规范
### 1. 坐标输入建议
如果您选择手动输入经纬度进行查询，请遵循以下 WGS84 坐标规范：
- **经度范围**: `0` 至 `360` (注：脚本会自动处理 `-180` 到 `180` 的负数经度并转换为对应格式)。
- **纬度范围**: `-90` 至 `90`。

### 2. 预测天数
目前版本建议预测天数在 **10 天** 以内（受 API 版本限制）。

### 3. 输出语言
- **主要语向**: 输出以**中文**为主。
- **动态适配**: 插件会根据您的 Claw 全局配置及 Agent 的上下文，自动将数据标签（如温度、风级）和统计总结翻译为用户所需的语种。
