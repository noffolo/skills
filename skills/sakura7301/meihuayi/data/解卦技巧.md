## 二、体用关系

> 体为求测者/主体，用为所问之事/客体

### 体用生克与吉凶

| 体用关系 | 吉凶 | 含义 | 建议 |
|----------|------|------|------|
| **体克用** | 吉 | 掌握主动权，事情向利己发展 | 积极推进 |
| **用克体** | 凶 | 被动受压，事情受阻 | 审时度势，必要时暂停 |
| **体生用** | 平中带耗 | 付出多于收获，劳心费力 | 评估性价比 |
| **用生体** | 吉 | 外部资源倾斜，有意外收获 | 把握机会 |
| **体用比和** | 大吉 | 双方和谐，做事顺遂 | 放手去做 |

### 体用生克的力量判断

1. **月令影响**：体卦旺则抗克能力强，体卦衰则抗克能力弱
2. **旺相之时**：体卦旺相，能克衰败之用卦
3. **休囚之时**：体卦休囚，难以克制旺盛之用卦

## 三、主互变三卦综合判断

### 主卦（本卦）
代表**当前状态**和**事情主体**，动爻所在位置是**变化的触发点**。

### 互卦
代表**事情发展过程**和**中间状态**，体现事情演变的中间环节。

### 变卦
代表事情**最终结果**和**变化后的状态**。

### 综合判断要点

1. **先看主卦体用关系** — 确定基本吉凶基调
2. **再看动爻变化** — 动爻是变化的转折点
3. **参考互卦** — 了解事情发展过程
4. **以变卦定结局** — 变卦是最终呈现的结果
5. **结合月令** — 五行旺衰影响吉凶程度

## 四、动爻分析

### 动爻位置含义

| 爻位 | 含义 |
|------|------|
| 初爻 | 事情刚开始，基础层面 |
| 二爻 | 内部阶段，方案层面 |
| 三爻 | 中期阶段，执行层面 |
| 四爻 | 接近目标，外部层面 |
| 五爻 | 关键位置，领导层面 |
| 上爻 | 最终结局，顶层层面 |

### 动变规则
动爻之阴阳发生改变，阳爻变阴爻，或阴爻变阳爻，从而产生变卦。

## 五、应期判断

> 应期判断是梅花易数最难的部分，需要结合体用、象数、实际情况综合判断

### 应期数字计算法

以动爻位置数 + 用卦数（或变卦卦数）之和，用月令或日辰起算。

**口诀**：
- 近事以日计
- 短期事以月计
- 长期事以年计

### 应期判断要点

1. **体用关系转化时** — 体用关系由凶转吉或由吉转凶的转折点
2. **用卦旺相之时** — 用卦当令或临长生时，事情容易发生
3. **冲合之时** — 逢冲或逢合的日期
4. **动爻逢值** — 动爻地支当值之日
5. **变卦应期** — 以变卦的卦象配合判断

### 实用应期判断

| 询问范围 | 应期参考 |
|----------|----------|
| 几日 | 1-10日 |
| 近期 | 一月之内 |
| 短期 | 三月之内 |
| 中期 | 半年之内 |
| 长期 | 一年之内或更久 |

## 六、断卦步骤

1. **确认问题** — 明确用户所问之事
2. **获取排盘** — 执行程序获取卦象
3. **分析体用** — 判断体用生克关系及旺衰
4. **解读动爻** — 分析动爻位置和变化
5. **综合三卦** — 主卦看现状，互卦看过程，变卦看结果
6. **判断应期** — 事情发生的大致时间
7. **综合结论** — 给出合理的推断和建议

## 七、常见事项取象要点

| 事项 | 重点卦象 | 关注点 |
|------|----------|--------|
| 财运 | 乾（金钱）、坤（积累）、坎（流动） | 体用生克，用生体则财来 |
| 感情 | 兑（少女）、离（中女）、震（长男） | 互卦情感变化，变卦最终结果 |
| 工作 | 乾（领导）、离（文书）、坎（险阻） | 体克用则掌主动，用克体则受阻 |
| 健康 | 对应八卦人体部位 | 某卦受克严重则对应部位有问题 |
| 出行 | 震（动）、巽（入）、坎（险） | 变卦吉则出行顺利，凶则生变 |
