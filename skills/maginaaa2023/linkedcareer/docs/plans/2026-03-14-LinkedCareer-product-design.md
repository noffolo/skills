# LinkedCareer 产品设计文档
## 项目愿景
让龙虾帮助用户规划和记录职业生涯，并找到志同道合的行业伙伴，降低职业交流的门槛。
## 核心定位
- 插件部分（LinkedCareer）：开源的本地职业生涯管理工具，支持OpenClaw/Claude Code/Codex平台，用户完全掌控自己的职业数据
- 社区部分（LinkedClaw）：AI Agent时代的职业社区，用户通过自己的Agent参与交流，打通职业成长和人脉链接
## P0 MVP 核心功能
### 1. 职业生涯记录模块
#### 1.1 初始化引导
- 首次使用时通过多轮对话引导用户完善基础信息
  - 基本信息：姓名、联系方式、所在城市、期望岗位等
  - 教育经历：学校、专业、学历、毕业时间、相关经历
  - 职业经历：每段工作的公司、岗位、入职/离职时间
- 自动追问机制：对每段职业经历自动追问以下内容
  - 核心负责的项目和产出业绩
  - 掌握的专业技能和能力提升
  - 主要工作成果和量化数据
- 简历导入功能：支持用户上传已有简历（文本/Markdown/Word/PDF），自动解析内容快速补全基础信息，减少用户输入成本
#### 1.2 定期记录提醒
- 支持自定义提醒频率：每日/每周/每月
- 自动推送引导问题，帮助用户记录：
  - 近期完成的工作项目和成果
  - 新掌握的技能和能力提升
  - 遇到的挑战和解决方案
  - 职业思考和规划调整
#### 1.3 本地数据存储
- 双格式存储，兼顾AI读取和用户可读
  - 结构化JSON：`career_data.json`，便于AI分析和处理
  - 可读Markdown：`career_records.md`，用户可以直接编辑和查看
- 数据完全存储在用户本地 `~/.openclaw/workspace/LinkedCareer/user_data/` 目录，用户完全可控，不上传任何数据到第三方服务器
### 2. 简历生成模块
#### 2.1 人岗匹配评估
- 输入目标岗位JD后，自动计算人岗匹配度（0-100分）
- 匹配度低于60分：明确提醒用户不建议投递该岗位，说明不匹配的核心原因
- 用户坚持投递：不会编造任何履历信息，仅基于已有内容做最优适配
#### 2.2 智能生成逻辑
- 两种生成模式：
  1. **通用生成**：基于所有职业经历生成通用版简历
  2. **定向生成**：针对具体公司、岗位、JD要求，优先展示匹配的技能、项目和经验
- 生成原则：完全基于用户真实记录，客观呈现，不编造、不夸大
#### 2.3 多模板支持
提供3套简历模板，用户可按需选择：
1. **极简版**：内容高度浓缩，排版紧凑，适合1页PDF，用于简历初筛
2. **适中版**：平衡信息密度和可读性，1-2页PDF，适合大部分面试场景
3. **详细版**：完整展示所有项目和经验，3-4页PDF，适合终面或深度职业沟通
#### 2.4 多格式导出
- 原生支持Markdown格式导出，无任何依赖，用户可直接编辑
- 自动支持PDF/Word导出：集成现有OpenClaw技能完成格式转换，用户无需额外配置
- 同步生成配套的求职信/自检信，突出个人优势和岗位匹配点
### 3. 兼容性
- 优先完美支持OpenClaw平台
- 兼容Claude Code、Codex平台，不使用OpenClaw独有API
## P1 后续规划
1. 对接LinkedClaw社区，支持用户一键发布简历、参与职业话题讨论
2. 支持职业发展路径规划和建议
3. 支持面试模拟和复盘功能
4. 支持多语言简历生成
## 核心原则
1. **数据主权**：所有用户数据完全存储在本地，用户100%可控，绝不收集任何用户数据
2. **完全开源**：插件代码完全开源在GitHub，接受社区贡献和审计
3. **模型无关**：不绑定任何AI模型，用户使用自己配置的模型
4. **隐私优先**：所有功能离线可用，不需要联网即可使用核心功能
