#### 上市发债企业基本信息

接口：`get_chain_bond_issuers`　
请求方式: **`POST`**　
tool_id: `get_chain_bond_issuers`

通过企业ID或企业名称（支持模糊匹配）查询上市发债企业的基本信息，包括企业统一社会信用代码、中英文名称、是否为发债公司、是否为上市公司以及相关机构ID等核心标识信息，用于快速识别和筛选符合条件的发债主体。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `companyName` | ✅ | string | 企业名称 | `广东粤运交通股份有限公司` |

**接口示例**

```bash
node scripts/call_api.js chain/bond-issuers --method POST companyName=广东粤运交通股份有限公司
```

#### 证券基础信息(产业链）

接口：`get_chain_sec_basic_info`　
请求方式: **`POST`**　
tool_id: `get_chain_sec_basic_info`

通过股票代码查询证券（股票）的基础信息，包括证券代码、证券中英文简称、所属公司的中英文名称及机构标准代码、上市市场代码及中英文名称、上市状态、上市日期（yyyy-MM-dd HH:mm:ss）和退市日期（yyyy-MM-dd HH:mm:ss），适用于获取证券的标识、所属主体及上市状态等核心元数据。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `stockCode` | ✅ | string | 股票代码 | `000001` |

**接口示例**

```bash
node scripts/call_api.js chain/sec-basic-info --method POST stockCode=000001
```

#### 行业信息(产业链）

接口：`get_chain_industry_info`　
请求方式: **`POST`**　
tool_id: `c`

通过行业代码或行业名称（支持模糊匹配）查询产业链行业分类信息，返回行业代码、名称、父级代码、所有祖先代码、分类体系（如CSF、CSRC等）及行业层级等核心数据，用于构建和分析产业链上下游关系及行业分类体系。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryName` | — | string | 行业名称【与industryCode组成多选多参数，必须传递其中一个至多个】 | `商品化工` |
| `industryCode` | — | string | 行业编码【与industryName组成多选多参数，必须传递其中一个至多个】 | `CSF_15101010` |

**接口示例**

```bash
# 可选参数: industryName, industryCode
node scripts/call_api.js chain/industry-info --method POST
```

#### 产品信息

接口：`get_chain_product_info`　
请求方式: **`POST`**　
tool_id: `get_chain_product_info`

通过产品代码或产品名称（支持模糊查询）查询供应链产品信息，返回产品代码、名称、所属行业代码、产品层级、直属父级产品代码以及所有祖先层级产品代码，用于构建和分析产品层级关系及行业归属。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `productCode` | — | string | 产品编码 | `P0007784` |
| `productName` | — | string | 产品名称 | `白茶` |

**接口示例**

```bash
# 可选参数: productCode, productName
node scripts/call_api.js chain/product-info --method POST
```
