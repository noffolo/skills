# team-building

Generate team activities, icebreakers, and random team assignments. Use when planning team events.

## Commands

### `icebreaker`

```bash
scripts/script.sh icebreaker
```

### `activity`

```bash
scripts/script.sh activity <size>
```

### `quiz`

```bash
scripts/script.sh quiz <topic>
```

### `random-team`

```bash
scripts/script.sh random-team <names count>
```

### `pairs`

```bash
scripts/script.sh pairs <file>
```

### `vote`

```bash
scripts/script.sh vote <options>
```

## Data Storage

Data stored in `~/.local/share/team-building/`.
